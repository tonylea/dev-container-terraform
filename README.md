# Devcontainer Image: Terraform

This repo creates a devcontainer image for use in development of Terraform projects.

## Built in Tools

The image contains the following tools:

| Name       | Version |
| ---------- | ------- |
| Terraform  | 1.3.7   |
| Terragrunt | 0.45.11 |
| tflint     | 0.46.1  |
| docker     | lts     |
| go         | lts     |
| node       | lts     |
| powershell | lts     |
